Windows Server 2019 Install

Posted Aug 26, 2022 Updated Apr 18, 2023
By Brian Biddle
3 min read

Active Directory Install

Today I will be going over my Windows Server installation. I will be using this as my domain controller by setting up Active Directory along with DNS for my local systems.

Proxmox setup

  1. To start it all off we will need the ISO for our install.
    1. You can download it here.
    2. This will only be able to be used for 180 days however I will outline a way to extend this timer if needed.
  2. Once downloaded you can either go bare metal or virtual. for me I will be installing this on Proxmox so for this I will need to upload the ISO to the server over the web interface.

  1. Now that it is uploaded proceed to making the VM.

1. Name: windows-server-19

2. ISO image: winserver2019.iso 3. Type: Microsoft Windows 4. Version: 10/2016/2019

5. Qemu Agent: check mark

6. Disk Size: 50gib 7. Bus/Device: SCSI 8. SCSI controller: Virtio SCSI 9. Cache: write back 10. Discard: check mark 11. Cores: 2 (this is all i needed) 12. Memory: 4096 (again all i needed) 13. Bridge: vmbr1 (this is my bridge for my Pfsense install within Proxmox.)

  1. Another thing we will need to do if installing on Proxmox is the virtio ISO drivers. this will improve compatibility and performance of windows when virtualized.
  2. Download the virtio drivers here.
    1. Upload the ISO to Proxmox.
    2. Now go to the hardware tab of the windows server VM and add a CD/DVD drive with virtio-win.iso selected.

  1. Now you are ready to boot.

Windows Install

  1. Start following the installation wizard up to which OS you want to install. I chose the desktop version due to simplicity of using a GUI but feel free to install just the server core for a smaller attack surface over all.

  1. Get to custom or upgrade.

  1. Select custom and new.

  1. Create
  2. Your install should be running now. This will take a while.
  3. Now to ensure all drivers are good head over to the device manager and click on anything with a driver issue.

  1. Click Update Driver.

  1. Select Browse my computer.

  1. Browse to the virtio disk and check the include sub folders box and run.

  1. Once all are good you have a working base install. ( if you are in a VM like me now would be a good time to take a snapshot or backup of the VM in case we mess something up in the future ;)

Install Active Directory & DNS

Now to install Active Directory and DNS on this server.

  1. Head over to server manager located in the start menu if it did not auto start.
  2. At the top right locate the manage button and select “ Add roles and features”.
  3. Select next up till you get to server roles.

  1. Select both “DNS” and “active directory domain services”.
  2. Continue and install.

  1. now on the main page of server manager locate the flag next to the manage button.
  2. you will see that the system needs to be promoted to a domain controller.
  3. select the hyperlink in the notification.

  1. Select add a new forest.
  2. Name it how you want.

  1. Keep going till you get to the prereq checks (ignore the DNS error).
  2. it should all pass and their will be a few errors however ultimately it should let you install.

  1. Once rebooted you now have a DNS and active directory domain controller!
  2. Next i will be going over how i configured them and connecting both a windows 10 and windows 11 systems to the domain in a later post.
